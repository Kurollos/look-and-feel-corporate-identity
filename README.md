# Look and Feel Embassy

## Beschrijving
Deze sprint draait om het kleuren- en designconcept van mijn site. Samen met mijn groepje heb ik een **styleguide** gemaakt en deze geïntegreerd in de site.  

De site bestaat uit de **homepage** en de **catalogs-pagina**:  

- **Homepage**:  
  - Navigatielinks bovenaan  
  - Kaartjes met linkjes en "Lees meer"  
  - Footer onderaan  

- **Catalogs-pagina**:  
  - Catalogusformulier  
  - Filterfunctie  
  - Kaarten voor catalogusitems  

Bekijk de site hier: [Look and Feel Embassy](https://kurollos.github.io/look-and-feel-corporate-identity/index.html)

## Kenmerken
- **HTML**: Gestructureerde pagina’s met semantische tags voor header, main, footer en kaarten.
